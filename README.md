# 🚀 TypeScript Template

Simple TypeScript template for Node.js projects.

## ✨ Features

- [Nodemon](https://nodemon.io/)
- [Prettier](https://prettier.io/)
- [tsup](https://tsup.egoist.dev)
- [tsx](https://github.com/privatenumber/tsx)

## 📦 Installation

```bash
pnpm install
```

## 🚀 Usage

```bash
pnpm dev
```

## 📦 Build

```bash
pnpm build
```
